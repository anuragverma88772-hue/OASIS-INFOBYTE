# Google Play Store Data Analytics

## Objective
Analyze Android app market trends using Google Play Store data.

## Dataset
- apps.csv
- user_reviews.csv

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

## Analysis Performed
✔ Data Cleaning

✔ Exploratory Data Analysis

✔ Category Distribution

✔ Rating Analysis

✔ Installs Analysis

✔ Price Analysis

✔ Content Rating Analysis

✔ Correlation Heatmap

✔ Top Installed Apps

✔ Top Rated Apps

✔ Sentiment Analysis

✔ Interactive Dashboard

## Key Insights

1. Family and Game categories contain the highest number of applications.
2. Highly rated apps generally receive more installs.
3. Most applications are free.
4. Reviews and installs have a strong positive correlation.
5. Positive reviews dominate user feedback.
6. Paid applications form a very small percentage of the Play Store.
